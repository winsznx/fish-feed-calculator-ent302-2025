## ğŸ“ Calculation Method

The calculator follows the **standard Pearson Square methodology**:

1. **Mean Calculation** â€“ Average of protein and energy sources (FM, SBM, GNC, YM, RB, Maize)
2. **Pearson Square** â€“ Determines optimal ratios based on target crude protein (CP)
3. **Proportional Distribution** â€“ Allocates ingredients by protein contribution
4. **Weight Calculation** â€“ Converts ratios to actual weights
5. **Cost Analysis** â€“ Computes total formulation cost
